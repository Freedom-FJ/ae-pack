## API

### RichText

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| height | 隐藏时的高度 | number/string | - |
| value | 控制组建展开还是收缩 | boolean | - |
| showBtn | 是否显示展开收缩按钮 | boolean | - |
| onChange | 收缩展开状态改变 | boolean | - |
